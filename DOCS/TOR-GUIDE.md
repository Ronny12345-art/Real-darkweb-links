# Understanding Tor Network

## What is Tor?
The Tor (The Onion Router) network is free software for enabling anonymous communication.

## How It Works
1. **Onion Routing**: Encrypted data passes through multiple nodes
2. **Layered Encryption**: Like layers of an onion
3. **Exit Nodes**: Final point where traffic leaves Tor network

## Legal Uses
- Journalist communications
- Whistleblower protections
- Privacy research
- Circumventing censorship
